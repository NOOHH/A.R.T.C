Common Issues & Solutions:
  • Email not received: Check spam folder, verify Gmail SMTP settings
  • SMTP errors: Ensure you're using a Gmail App Password, not regular password
  • CSRF errors: Clear browser cache and cookies
  • Route errors: Run php artisan config:clear and php artisan route:clear
Gmail SMTP Requirements:
  • Enable 2-Factor Authentication on your Gmail account
  • Generate an App Password specifically for this application
  • Use the App Password in MAIL_PASSWORD, not your regular Gmail password
